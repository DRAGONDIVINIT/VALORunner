5.游戏介绍
游戏名:
《瓦罗兰特跑酷之向智能科学奔去》
游戏版本:
开发版本
游戏简介:
你说的对，但是《瓦罗兰特跑酷之向智能科学奔去》是由南京大学技术科学试验班曾闻天自主研发的一款全新跑酷游戏。游戏发生在一个被称作「南京大学」的幻想世界，在这里，被神选中的人将被授予「学分」，导引绩点之力。你将扮演两位名为「瓦罗兰特特工」的神秘角色，在自由的旅行中邂逅蚂蚁、撒酱(没有实装)，和他们一起击败障碍，夺得更多的学分和更高的绩点——同时，逐步发掘「智能科学」的真相。
游戏功能:
主角不断前进且随着前进画面在刷新
玩家可以看见共计18行的道路
分数(绩点)随时间增加,且分数增加会加快画面刷新速度,最高速度在200分时达到300ms的刷新速度
游戏开始显示主页面,根据提示可进入规则界面,商店界面,角色选择界面或直接结束游戏
进入规则界面后可查看游戏操作规则,在规则界面可进入角色选择界面或直接结束游戏
进入商店界面可支付100学分(游戏内货币)购买[猎枭]和[不死鸟]两个角色,也可进入角色选择界面或规则界面或直接结束游戏
进入角色选择界面后可选择你已有的角色开始游戏
游戏界面,"|"表示道路边界," "表示可通行区域,"U"表示上跳障碍物,"D"表示滑铲障碍物,"X"表示不可通行障碍,"@"表示可拾取道具[蚂蚁],"Y"表示可拾取"表示人物
在道路下方显示当前数据,包括:
本局游戏已获得学分,当前绩点,玩家状态(正常/跳跃/滑铲),技能充能情况(若角色为白板作者君则显示"你是白板")
在游戏过程中,通过"a"和"d"控制人物左右移动
通过"w"和"s"控制角色跳跃和滑铲,此时角色会变为"^"或"v",可分别通行"U"和"D"
通过" "实现暂停游戏,可按键盘上任意键(不包括电源键)结束暂停
当充能达到50后,不再继续充能,此时可以通过"c"释放角色技能
当人物拾取"Y"时,学分+1
当人物拾取"@"时,会清除所有生成物,同时视为拾取了当时地图上所有"Y"
当人物不在对应状态接触"U""D"或"X"时,游戏结束
障碍物生成基本保证玩家总是能继续前进的
目前实装角色包括:
猎枭:技能[我,是猎人!],开启后清除面前一排的生成物,并得到所清除的生成物数量的学分
不死鸟:技能[来吧,走起!],开启后,在前方十格获得无敌,不会因碰撞障碍死亡
白板作者君:和飞舞作者一样是个飞舞
每次刚运行游戏时,猎枭与不死鸟默认锁定,只能选择白板作者君进行游戏
不结束游戏程序的话,每局游戏所获得的学分可积累
